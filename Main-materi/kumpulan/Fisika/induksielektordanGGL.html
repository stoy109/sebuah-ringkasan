Induksi Elektromagnetik dan GGL

Faktor yang Mempengaruhi GGL

• Kecepatan perubahan fluks magnetik - semakin cepat perubahan, semakin besar GGL

• Jumlah lilitan loop/kumparan - semakin banyak lilitan, semakin besar GGL

Hukum Faraday

GGL induksi dinyatakan dengan rumus:

\[ \varepsilon = N \frac{\Delta \phi}{\Delta t} \]

Dimana:

ε = GGL (Volt)

N = Jumlah lilitan

Δφ/Δt = Perubahan fluks magnetik per waktu (Wb/s)

GGL Maksimum

\[ \varepsilon_m = NBA\omega \]

Dimana:

N = Jumlah lilitan

B = Medan magnet (Tesla)

A = Luas loop (m²)

ω = Kecepatan sudut (rad/s)

Persamaan Umum GGL

\[ \varepsilon = \varepsilon_m \sin \omega t \]

Catatan Penting:

• Konversi luas: cm² ke m² × 10⁻⁴

• GGL maksimum saat sudut 90° (sin 90° = 1)

• GGL nol saat sudut 0° (sin 0° = 0)
