Geometri Analitik: Lingkaran

Pengertian Lingkaran

Lingkaran adalah tempat kedudukan titik-titik pada bidang datar yang berjarak sama terhadap suatu titik tertentu. Titik tertentu ini disebut sebagai pusat lingkaran, dan jarak dari pusat ke setiap titik pada lingkaran disebut jari-jari.

Persamaan Lingkaran

1. Persamaan Umum Lingkaran

Jika pusat lingkaran berada di titik (a,b) dengan jari-jari r, maka persamaan umumnya adalah:

(x-a)² + (y-b)² = r²

2. Persamaan Lingkaran dengan Pusat (0,0)

x² + y² = r²

3. Persamaan Bentuk Umum

x² + y² + Dx + Ey + F = 0

Pusat: (-D/2, -E/2)

Jari-jari: r = √((D/2)² + (E/2)² - F)

Kedudukan Titik dan Garis Terhadap Lingkaran

1. Kedudukan Titik P(x,y):

• (x-a)² + (y-b)² < r² : titik P di dalam lingkaran

• (x-a)² + (y-b)² = r² : titik P pada lingkaran

• (x-a)² + (y-b)² > r² : titik P di luar lingkaran

2. Kedudukan Garis y = mx + c:

D = (2mc)² - 4(1+m²)(c²-r²)

• D < 0 : garis tidak memotong lingkaran

• D = 0 : garis menyinggung lingkaran

• D > 0 : garis memotong lingkaran di dua titik
